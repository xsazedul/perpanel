# jtg panel

Made by Jishnu

## Installation

1. Clone the repository:
   ```bash
   git clone <repository-url>
   cd jtg-panel
   ```

2. Install dependencies:
   ```bash
   npm install
   ```

3. Build the application:
   ```bash
   npm run build
   ```

4. Start the server:
   ```bash
   npm run start
   ```

## Development

To run the panel in development mode with auto-reloading:

```bash
npm run dev
```
